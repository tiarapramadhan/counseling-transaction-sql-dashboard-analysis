## Isi Dashboard

- **Revenue by Package Type** — bar chart revenue per jenis paket (konseling, call, video, meditasi).
- **Revenue by Cluster Type** — bar chart revenue per jenis cluster (call, general, video, text).
- **Top Users by Spending** — tabel ranking user berdasarkan total spend (grandTotal).
- **Transactions by Package Type** — jumlah transaksi per jenis paket.
- **Transaction Status Distribution** — donut chart proporsi status transaksi (success, cancel, expire, failed, pending).
- **Filter Interaktif** — rentang tanggal, status, counselingType, packageType.

## Insight Utama

- Paket konseling menghasilkan revenue tertinggi di antara semua jenis paket.
- Mayoritas transaksi yang tercatat berstatus sukses (74,8%).
- Distribusi revenue cukup bervariasi antar jenis paket maupun jenis konseling.
